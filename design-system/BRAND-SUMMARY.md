# X-Ray Tech — brand summary

## What it is

A credential-tracking app for X-ray technologists and the imaging managers who lead them. It turns the question *"am I on track with my CE points and license renewals?"* into a single live number on one screen — for the individual tech and for the manager looking at the whole team.

## Why it exists

Built originally with Michelle, an imaging manager at a Chicago-area hospital. She was tracking her team's continuing-education credits in a Google Sheet across four state-board websites. Every manager she'd ever talked to was doing some version of the same thing. X-Ray Tech collapses that into one screen — so nobody finds out about a lapsed license 30 days too late.

It is definitely *not*: **basic** or **cookie-cutter**.

## Archetype + anti-archetype

- **Archetype**: *The Quiet Steward.* Holds your records the way a great librarian holds a borrower's history — accurately, without theater, ready when asked.
- **Anti-archetype**: *The Cheerleader.* No confetti. No "🎉 You did it!" toasts. The number going up is the reward.

## Visual foundations — snapshot

| Aspect       | Choice                                                                                      |
| ------------ | ------------------------------------------------------------------------------------------- |
| Primary      | `brand/600` = `#5B3FE4` (electric violet)                                                    |
| Heritage     | `ink-purple` = `#4E2A84` (legacy royal purple, retained for headings)                        |
| Canvas       | `paper` = `#FFFFFF` for marketing, `surface` = `#FAFAFC` for dashboard                        |
| Ink          | `#0E0B1F` (deep cool near-black with violet undertone)                                       |
| Headings     | **Geist** 600, letter-spacing −0.015 to −0.02em                                              |
| Body         | **Inter** 400/500, normal letter-spacing                                                     |
| Numbers      | **JetBrains Mono** — license points, days-until-expiry, cert IDs, dates                      |
| Radius       | `md` 8px on dashboard cards; `full` pill on marketing CTAs; `sm` 6px on product UI buttons   |
| Icons        | Lucide, 1.5px stroke, outline only, `currentColor`                                            |
| Logo motif   | Radial iris / aperture — outer ring + six radiating emission blades + central pupil. Reads as an X-ray lens, an emission pattern, and a precision instrument. |
| Motion       | Fade + small translate. 160 / 240 / 480ms durations. No parallax, no looping video.          |
| Imagery      | Real photographs of real X-ray techs in real imaging suites — never stock or AI-generated     |

## Voice in one paragraph

X-Ray Tech sounds like a competent imaging manager talking to one of her techs — direct, short sentences, second-person singular, no hype, no SaaS jargon. We speak the technologist's actual licensing vocabulary (ARRT, IEMA, CE points, Category A vs. B, modality). Every claim has a number or a name behind it: not "save time" but *"saves 2 hours per Monday."* We never write a sentence we could ship for a competitor by changing the brand name.

## What we want someone to feel in 5 seconds

"This was built by someone who knows the difference between Category A and Category B CE credits. Finally a tool that speaks my actual job — not a generic compliance portal."
